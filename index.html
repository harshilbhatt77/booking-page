Action Required: Setup Google Calendar API

To use this booking page, you need to get your own Google API Key and Client ID.

  1. Go to the Google Cloud Console.
  2. Create a new project.
  3. Go to "APIs & Services" > "Library", search for and enable the **Google Calendar API**.
  4. Go to "APIs & Services" > "Credentials".
  5. Click "+ CREATE CREDENTIALS" and create an **API Key**. Copy it.
  6. Click "+ CREATE CREDENTIALS" again and create an **OAuth client ID**.
    • Select "Web application" as the type.
    • Under "Authorized JavaScript origins", add the URL where you will host this page.
  7. Copy the **Client ID**.
  8. Paste your API Key and Client ID into the script at the top of this file.
